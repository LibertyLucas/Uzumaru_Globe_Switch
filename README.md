# VPS 出口切换 Telegram Bot

**一个轻量级 Telegram 机器人**，通过 SSH 远程控制你的 VPS，实现**一键切换出口节点**、查看当前 IP、批量检测所有节点可用性等功能。

专为需要频繁切换出口（如 Uzmaru / 多线路 VPS）用户设计，带详细日志和 Markdown 美化输出。

---

## 📋 功能特性

- `/nm` 一键获取节点列表并通过按钮切换出口
- `/ip` 查看当前出口 IP 的完整信息（国家、城市、ISP 等）
- `/check` 批量检测所有出口节点连通性（自动恢复原始节点）
- `/status` 检查 VPS SSH 连接状态
- `/start` & `/help` 欢迎与详细说明
- 详细日志记录（`bot.log`）
- 支持授权用户列表（防止他人滥用）
